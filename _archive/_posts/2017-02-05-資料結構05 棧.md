## 1、棧是什麼

棧是一種特殊的線性表，它限定了只能在表的一端進行插入與刪除操作。因此，棧就是後進先出 Last In First Out (LIFO) 的線性表。

線性表分為順序表和連結串列，所以棧分為**順序棧**和**鏈式棧**，為了方便演示，下文所使用的棧都是**順序棧**。 

![][1]

它的應用場景有很多，比如手槍彈夾裡面的子彈，只能從一端塞入或者取出子彈。

順序棧的程式碼實現：

*SeqStack.java*

<pre>/**
 * 自己封裝一個順序棧
 */
public class SeqStack {
    // 使用陣列來存放元素
    public Object[] data;

    // 棧頂指標
    public int top;

    public SeqStack(int length) {
        data = new Object[length];
    }
}</pre>

## 3、棧的常見操作及程式碼實現

### 3-1、初始化

實現思路：用指定大小的 length 例項化一個 SeqStack，然後使top指標指向-1

程式碼：

SeqStackOperate.java

<pre>public class SeqStackOperate {
    /**
     * 初始化
     * @param length
     * @return
     */
    public SeqStack init(int length) {
        SeqStack seqStack = new SeqStack(length);
        seqStack.top = -1;
        return seqStack;
    }
}</pre>

### 3-2、進棧

實現思路：將top指標加1，然後將新元素插入到top指標指向的位置。

程式碼：

SeqStackOperate.java 中新增方法

<pre>    /**
     * 進棧
     * @param seqStack
     * @param data
     */
    public void push(SeqStack seqStack, Object data) {
        // 檢查棧是否滿了
        if (seqStack.top == seqStack.data.length - 1) {
            return;
        }
        seqStack.top++;
        seqStack.data[seqStack.top] = data;
    }</pre>

### 3-3、出棧

實現思路：刪除top指標指向的元素，並使top指標減1

程式碼：

SeqStackOperate.java 中新增方法

<pre>    /**
     * 出棧
     * @param seqStack
     * @return
     */
    public Object pop(SeqStack seqStack) {
        // 檢查棧是否為空
        if (seqStack.top == -1) {
            return null;
        }
        Object obj = seqStack.data[seqStack.top];
        seqStack.data[seqStack.top] = null;
        seqStack.top--;
        return obj;
    }</pre>

### 3-4、獲取棧頂元素

實現思路：與出棧相似，但是不刪除棧頂元素。

程式碼：

SeqStackOperate.java 中新增方法

<pre>    /**
     * 獲取棧頂元素
     *
     * @param seqStack
     * @return
     */
    public Object getTopData(SeqStack seqStack) {
        // 檢查棧是否為空
        if (seqStack.top == -1) {
            return null;
        }
        return seqStack.data[seqStack.top];
    }</pre>

### 3-5、判斷是否棧空

實現思路：判斷top指標是否等於-1就可以

程式碼：

SeqStackOperate.java 中新增方法

<pre>    /**
     * 判斷是否棧空
     * @param seqStack
     * @return
     */
    public boolean isEmpty(SeqStack seqStack) {
        return seqStack.top == -1;
    }</pre>

### 3-6、判斷是否棧滿

實現思路：檢查top指標的值與陣列長度減1是否相等，如果相等則為棧滿。

程式碼：

SeqStackOperate.java 中新增方法

<pre>    /**
     * 判斷是否棧滿
     * @param seqStack
     * @return
     */
    public boolean isFull(SeqStack seqStack) {
        return seqStack.top == seqStack.data.length - 1;
    }</pre>

### 3-7、獲取棧中元素個數

實現思路：top指標的值加1就是棧中的元素個數。

程式碼：

SeqStackOperate.java 中新增方法

<pre>    /**
     * 獲取棧中元素個數
     *
     * @param seqStack
     * @return
     */
    public int getLength(SeqStack seqStack) {
        return seqStack.top + 1;
    }</pre>

## 4、測試

新增一個用來測試的類：StackTest.java

<pre>public class StackTest {

    public static void main(String[] args) {
        SeqStackOperate seqStackOperate = new SeqStackOperate();

        // 初始化一個棧，最大長度為5
        SeqStack seqStack = seqStackOperate.init(5);

        // 看看能否放進6個元素
        seqStackOperate.push(seqStack, 1);
        seqStackOperate.push(seqStack, 2);
        seqStackOperate.push(seqStack, 3);
        seqStackOperate.push(seqStack, 4);
        seqStackOperate.push(seqStack, 5);
        seqStackOperate.push(seqStack, 6);

        // 輸出棧當前的長度
         int length = seqStackOperate.getLength(seqStack);
        System.out.println("棧當前的長度：" + length);
        System.out.println("");

        // 出棧
        Object obj = seqStackOperate.pop(seqStack);
        System.out.println("出棧的元素是 ---- " + obj);
        System.out.println("");

        // 輸出棧當前的長度
        length = seqStackOperate.getLength(seqStack);
        System.out.println("棧當前的長度：" + length);
        System.out.println("");

        // 輸出當前的棧頂元素
        obj = seqStackOperate.getTopData(seqStack);
        System.out.println("當前的棧頂元素是 ---- " + obj);
        System.out.println("");
    }

}</pre>
