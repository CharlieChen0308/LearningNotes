二分查詢屬於順序表查詢，二分查詢也稱為折半查詢。二分查詢的時間複雜度為O(log<sub>2</sub>n)  

## 1、二分查詢的定義

什麼是二分查詢呢？二分查詢的基本思想是：在有序表中，取中間元素作為比較物件，若給定值與中間元素相等，則查詢成功；若給定值小於中間元素，則在中間元素的左半區繼續查詢；若給定值大於中間元素，則在中間元素的右半區繼續查詢。不斷重複上述過程，直到找到為止。

從二分查詢的定義我們可以看出，使用二分查詢有兩個前提條件：

（1）待查詢的列表必須有序。

（2）必須使用線性表的順序儲存結構來儲存資料。

## 2、二分查詢的程式碼實現

程式碼：

BinarySearch.java

<pre>public class BinarySearch {
    public static void main(String[] args) {
        int[] list = {10, 20, 30, 40, 50, 60, 70, 80, 90};
        System.out.println("************二分查詢************");
        display(list);
        System.out.println("");

        int result = binarySearch(list, 30);
        if (result != -1) {
            System.out.println("30在列表中的位置是：" + result);
        } else {
            System.out.println("對不起，列表中不存在該元素！");
        }
    }

    /**
     * 二分查詢
     */
    public static int binarySearch(int[] list, int key) {
        int low = 0;
        int high = list.length - 1;

        while (low <= high) {
            int middle = (low + high) / 2;

            // 判斷中間元素是否與給定值相等
            if (list[middle] == key) {
                return middle;
            } else {
                if (list[middle] > key) {
                    // 在中間元素的左半區查詢
                    high = middle - 1;
                } else {
                    // 在中間元素的右半區查詢
                    low = middle + 1;
                }
            }
        }
        // 沒有找到（查詢失敗）
        return -1;
    }

    /**
     * 遍歷列印
     */
    public static void display(int[] list) {
        System.out.println("********展示開始********");
        if (list != null && list.length > 0) {
            for (int num :
                    list) {
                System.out.print(num + " ");
            }
            System.out.println("");
        }
        System.out.println("********展示結束********");
    }
}</pre>
