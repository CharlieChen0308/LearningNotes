**題目：劃分一個整形陣列，把負數放在左邊，零放在中間，正數放在右邊**

思路：掃描遍歷，發現正數出現在負數前面，就調換位置。

程式碼：

```java
    public static int[] arrayChange(int[] array) {
        int length = array.length;
        int low = 0;
        int high = length - 1;

        // 掃描遍歷，發現正數出現在負數前面，就調換位置
        while (low < high) {
            while (array[low] < 0 && low < high) {
                low++;
            }
            while (array[high] > 0 && low < high) {
                high--;
            }
            int temp = array[low];
            array[low] = array[high];
            array[high] = temp;
        }
        return array;
    }
```
