連結串列類：*MyLinkedList.java*

```java
/**
 * 自己定義的一個連結串列類
 */
public class MyLinkedList {

    // 頭節點
    private Node headNode;
    // 用來遍歷連結串列的臨時節點
    private Node tempNode;

    // Getter
    public Node getHeadNode() {
        return headNode;
    }
    // Setter
    public void setHeadNode(Node headNode) {
        this.headNode = headNode;
    }

    // 連結串列的初始化方法
    public MyLinkedList() {
        // 初始化時，連結串列裡面只有1個節點，所以這個節點的地址域為null
        Node node = new Node("王重陽", null);
        // 頭節點不儲存資料，它的資料域為null，它的地址域儲存了第1個節點的地址
        headNode = new Node(null, node);
    }

    /**
     * 1、插入節點：時間複雜度為O(n)
     * @param newNode 需要插入的新節點
     * @param position 這個變數的範圍可以從0到連結串列的長度，注意不要越界。
     *                 頭節點不算進連結串列的長度，
     *                 所以從頭節點後面的節點開始算起，position為0
     */
    public void insert(Node newNode, int position) {
        // 透過position變數，讓tempNode節點從頭節點開始遍歷，移動到要插入位置的前一個位置
        tempNode = headNode;
        int i = 0;
        while (i < position) {
            tempNode = tempNode.next;
            i++;
        }
        newNode.next = tempNode.next;
        tempNode.next = newNode;
    }

    /**
     * 2、刪除節點：時間複雜度為O(n)
     * @param position
     */
    public void delete(int position) {
        // 這裡同樣需要用tempNode從頭開始向後查詢position
        tempNode = headNode;
        int i = 0;
        while (i < position) {
            tempNode = tempNode.next;
            i++;
        }
        tempNode.next = tempNode.next.next;
    }

    /**
     * 3、查詢節點：時間複雜度為O(n)
     * @param position
     * @return 返回查詢的節點
     */
    public Node find(int position) {
        // 這裡同樣需要用tempNode從頭開始向後查詢position
        tempNode = headNode;
        int i = 0;
        while (i < position) {
            tempNode = tempNode.next;
            i++;
        }
        return tempNode.next;
    }

    /**
     * 4、獲取連結串列的長度：時間複雜度為O(n)
     * @return
     */
    public int size() {
        tempNode = headNode.next;
        int size = 0;
        while (tempNode.next != null) {
            size = size + 1;
            tempNode = tempNode.next;
        }
        size = size + 1; // tempNode的地址域為null時，size記得加上最後一個節點
        return size;
    }

    // 遍歷連結串列，列印出所有節點的方法
    public void showAll() {
        tempNode = headNode.next;
        while (tempNode.next != null) {
            System.out.println(tempNode.element);
            tempNode = tempNode.next;
        }
        System.out.println(tempNode.element);
    }
}
```

## 1、從尾到頭列印單向連結串列

對於這種顛倒順序列印的問題，我們應該就會想到棧，**後進先出**。因此這一題**要麼自己新建一個棧，要麼使用系統的棧**（系統遞迴呼叫方法時的棧）。需要把連結串列遍歷完一次，所以它的時間複雜度為 O(n)

注意：不能先把連結串列反轉，再遍歷輸出，因為這樣做會破壞連結串列節點原來的順序。

### 方法1：自己新建一個棧

*QuestionOneDemo.java*

```java
import org.junit.Test;

import java.util.Stack;

public class QuestionOneDemo {

    /**
     * 從尾到頭列印單向連結串列
     * 方法1：自己新建一個棧
     *
     * @param head 引數為連結串列的頭節點
     */
    public void reversePrint(Node head) {

        // 判斷連結串列是否為空
        if (head == null) {
            return;
        }

        // 新建一個棧
        Stack<Node> stack = new Stack<Node>();

        // 用來遍歷的臨時節點，從頭節點開始
        Node tempNode = head;

        // 從頭節點開始遍歷連結串列，將除了頭節點之外的所有節點壓棧
        while (tempNode.getNext() != null) {
            tempNode = tempNode.getNext();
            stack.push(tempNode);
        }

        // 將棧中的節點列印輸出即可
        while (stack.size() > 0) {
            // 出棧操作
            Node node = stack.pop();
            System.out.println(node.getElement());
        }
    }

    /**
     * 用來測試的方法
     */
    @Test
    public void test() {
        MyLinkedList myLinkedList = new MyLinkedList();

        Node newNode1 = new Node("歐陽鋒", null);
        Node newNode2 = new Node("黃藥師", null);
        Node newNode3 = new Node("洪七公", null);

        myLinkedList.insert(newNode1, 1);
        myLinkedList.insert(newNode2, 2);
        myLinkedList.insert(newNode3, 3);

        System.out.println("----原連結串列 start----");
        myLinkedList.showAll();
        System.out.println("----原連結串列 end----");
        System.out.println("");

        System.out.println("====從尾到頭列印連結串列 start====");
        reversePrint(myLinkedList.getHeadNode());
        System.out.println("====從尾到頭列印連結串列 end====");
        System.out.println("");

        System.out.println("----原連結串列(依然保留了原來的順序) start----");
        myLinkedList.showAll();
        System.out.println("----原連結串列(依然保留了原來的順序) end----");

    }
}
```

### 方法2：使用系統的棧（遞迴）

```java
/**
     * 從尾到頭列印單向連結串列
     * 方法2：自己新建一個棧
     *
     * @param head 引數為連結串列的頭節點
     */
    public void reversePrint2(Node head) {

        // 判斷傳進來的引數節點是否為空
        if (head == null) {
            return;
        }

        // 遞迴
        reversePrint2(head.next);
        System.out.println(head.getElement());
    }
```

### 總結：  

方法2是基於遞迴實現的，程式碼看起來更簡潔，但有一個問題：當連結串列很長的時候，就會導致方法呼叫的層級很深，有可能造成棧溢位。而方法1是自己新建一個棧，使用迴圈壓棧和迴圈出棧，程式碼的穩健性要更好一些。

## 2、查詢單向連結串列中的倒數第k個節點

###  2-1：普通思路

先將整個連結串列從頭到尾遍歷一次，計算出連結串列的長度size，得到連結串列的長度之後，就好辦了，直接輸出第 size-k 個節點就可以了（注意連結串列為空，k為0，k大於連結串列中節點個數的情況）。因為需要遍歷兩次連結串列，所以時間複雜度為 T(2n) = O(n)    

程式碼如下：

QuestionTwoDemo.java

```java
import org.junit.Test;

public class QuestionTwoDemo {

    /**
     * 查詢連結串列中的倒數第k個節點的方法
     *
     * @param myLinkedList 需要查詢的連結串列作為引數傳遞進來
     * @param k            代表倒數第k個節點的位置
     * @return
     */
    public Node reciprocalFindNode(MyLinkedList myLinkedList, int k) throws Exception {
        int size = 0;

        // 如果頭節點為null，說明連結串列為空
        if (myLinkedList.getHeadNode() == null) {
            throw new Exception("連結串列為空");
        }

        // 判斷k，k不能為0
        if (k == 0) {
            throw new Exception("k不能為0");
        }

        // 第一次遍歷，計算出連結串列的長度size
        Node tempNode = myLinkedList.getHeadNode();
        while (tempNode != null) {
            size++;
            tempNode = tempNode.getNext();
        }

        // 判斷k，k不能大於連結串列中節點的個數
        if (k > size) {
            throw new Exception("k不能大於連結串列中節點的個數");
        }

        // 第二次遍歷，找出倒數第k個節點
        tempNode = myLinkedList.getHeadNode();
        for (int i = 0; i < size - k; i++) {
            tempNode = tempNode.getNext();
        }
        return tempNode;
    }

    /**
     * 用來測試的方法
     */
    @Test
    public void test() throws Exception {
        MyLinkedList myLinkedList = new MyLinkedList();

        Node newNode1 = new Node("歐陽鋒", null);
        Node newNode2 = new Node("黃藥師", null);
        Node newNode3 = new Node("洪七公", null);

        myLinkedList.insert(newNode1, 1);
        myLinkedList.insert(newNode2, 2);
        myLinkedList.insert(newNode3, 3);

        System.out.println("-----完整的連結串列start-----");
        myLinkedList.showAll();
        System.out.println("-----完整的連結串列end-------");
        System.out.println("");

        Node node = reciprocalFindNode(myLinkedList, 1);
        System.out.println("連結串列的倒數第1個節點是：" + node.getElement());

        node = reciprocalFindNode(myLinkedList, 2);
        System.out.println("連結串列的倒數第2個節點是：" + node.getElement());

        node = reciprocalFindNode(myLinkedList, 3);
        System.out.println("連結串列的倒數第3個節點是：" + node.getElement());

        node = reciprocalFindNode(myLinkedList, 4);
        System.out.println("連結串列的倒數第4個節點是：" + node.getElement());
    }
}
```

###  2-2：改進思路

這裡需要用到兩個節點型的變數：即變數 first 和 second，首先讓 first 和 second 都指向第一個節點，然後讓 second 節點往後挪 k-1 個位置，此時 first 和 second 就間隔了 k-1 個位置，然後整體向後移動這兩個節點，**直到 second 節點走到最後一個節點時，此時 first 節點所指向的位置就是倒數第k個節點的位置**。時間複雜度為O(n)

```java
     * 查詢連結串列中的倒數第k個節點的方法2
     *
     * @param myLinkedList 需要查詢的連結串列作為引數傳遞進來
     * @param k            代表倒數第k個節點的位置
     * @return
     */
    public Node reciprocalFindNode2(MyLinkedList myLinkedList, int k) throws Exception {
        // 如果頭節點為null，說明連結串列為空
        if (myLinkedList.getHeadNode() == null) {
            throw new Exception("連結串列為空");
        }

        Node first = myLinkedList.getHeadNode();
        Node second = myLinkedList.getHeadNode();

        // 讓second節點往後挪 k-1 個位置
        for (int i = 0; i < k - 1; i++) {
            second = second.getNext();
        }

        // 讓first節點和second節點整體向後移動，直到second節點走到最後一個節點
        while (second.getNext() != null) {
            first = first.getNext();
            second = second.getNext();
        }
        // 當second節點走到最後一個節點時，first節點就是我們要找的節點
        return first;
    }
```

## 3、反轉一個單向連結串列

例如連結串列：

1 -> 2 -> 3 -> 4

反轉之後：

4 -> 3 -> 2 -> 1

思路：

從頭到尾遍歷原連結串列的節點，每遍歷一個節點，將它放在新連結串列（實際上並沒有建立新連結串列，這裡用新連結串列來描述只是為了更方便的理解）的最前端。 時間複雜度為O(n)   

（注意連結串列為空和只有一個節點的情況）

```java
import org.junit.Test;

public class QuestionThreeDemo {

    /**
     * 反轉一個單向連結串列的方法
     *
     * @param myLinkedList
     * @throws Exception
     */
    public void reverseList(MyLinkedList myLinkedList) throws Exception {
        // 判斷連結串列是否為null
        if (myLinkedList == null || myLinkedList.getHeadNode() == null || myLinkedList.getHeadNode().getNext() == null) {
            throw new Exception("連結串列為空");
        }

        // 判斷連結串列裡是否只有一個節點
        if (myLinkedList.getHeadNode().getNext().getNext() == null) {
            // 連結串列裡只有一個節點，不用反轉
            return;
        }

        // tempNode 從頭節點後面的第一個節點開始往後移動
        Node tempNode = myLinkedList.getHeadNode().getNext();
        // 當前節點的下一個節點
        Node nextNode = null;
        // 反轉後新連結串列的頭節點
        Node newHeadNode = null;

        // 遍歷連結串列，每遍歷到一個節點都把它放到連結串列的頭節點位置
        while (tempNode.getNext() != null) {
            // 把tempNode在舊連結串列中的下一個節點暫存起來
            nextNode = tempNode.getNext();
            // 設定tempNode在新連結串列中作為頭節點的next值
            tempNode.setNext(newHeadNode);
            // 更新newHeadNode的值，下一次迴圈需要用
            newHeadNode = tempNode;
            // 更新頭節點
            myLinkedList.setHeadNode(newHeadNode);
            // tempNode往後移動一個位置
            tempNode = nextNode;
        }
        // 舊連結串列的最後一個節點的next為null，要把該節點的next設定為新連結串列的第二個節點
        tempNode.setNext(newHeadNode);
        // 然後把它放到新連結串列的第一個節點的位置
        myLinkedList.setHeadNode(tempNode);
        // 新建一個新連結串列的頭節點
        newHeadNode = new Node(null, tempNode);
        myLinkedList.setHeadNode(newHeadNode);
    }

    /**
     * 用來測試的方法
     */
    @Test
    public void test() throws Exception {
        MyLinkedList myLinkedList = new MyLinkedList();

        Node newNode1 = new Node("歐陽鋒", null);
        Node newNode2 = new Node("黃藥師", null);
        Node newNode3 = new Node("洪七公", null);

        myLinkedList.insert(newNode1, 1);
        myLinkedList.insert(newNode2, 2);
        myLinkedList.insert(newNode3, 3);

        System.out.println("-----完整的連結串列start-----");
        myLinkedList.showAll();
        System.out.println("-----完整的連結串列end-------");
        System.out.println("");

        System.out.println("-----反轉之後的連結串列start-----");
        reverseList(myLinkedList);
        myLinkedList.showAll();
        System.out.println("-----反轉之後的連結串列end-------");
    }
}
```

連結串列類：MyNumberLinkedList.java

```java
public class MyNumberLinkedList {
    // 頭節點
    private NumberNode headNode;
    // 用來遍歷連結串列的臨時節點
    private NumberNode tempNode;

    // Getter
    public NumberNode getHeadNode() {
        return headNode;
    }
    // Setter
    public void setHeadNode(NumberNode headNode) {
        this.headNode = headNode;
    }

    // 連結串列的初始化方法
    public MyNumberLinkedList() {
        // 初始化時，連結串列裡面只有1個節點，所以這個節點的地址域為null
        NumberNode node = new NumberNode(0, null);
        // 頭節點不儲存資料，它的資料域為null，它的地址域儲存了第1個節點的地址
        headNode = new NumberNode(null, node);
    }

    /**
     * 1、插入節點：時間複雜度為O(n)
     * @param newNode 需要插入的新節點
     * @param position 這個變數的範圍可以從0到連結串列的長度，注意不要越界。
     *                 頭節點不算進連結串列的長度，
     *                 所以從頭節點後面的節點開始算起，position為0
     */
    public void insert(NumberNode newNode, int position) {
        // 透過position變數，讓tempNode節點從頭節點開始遍歷，移動到要插入位置的前一個位置
        tempNode = headNode;
        int i = 0;
        while (i < position) {
            tempNode = tempNode.next;
            i++;
        }
        newNode.next = tempNode.next;
        tempNode.next = newNode;
    }

    /**
     * 2、刪除節點：時間複雜度為O(n)
     * @param position
     */
    public void delete(int position) {
        // 這裡同樣需要用tempNode從頭開始向後查詢position
        tempNode = headNode;
        int i = 0;
        while (i < position) {
            tempNode = tempNode.next;
            i++;
        }
        tempNode.next = tempNode.next.next;
    }

    /**
     * 3、查詢節點：時間複雜度為O(n)
     * @param position
     * @return 返回查詢的節點
     */
    public NumberNode find(int position) {
        // 這裡同樣需要用tempNode從頭開始向後查詢position
        tempNode = headNode;
        int i = 0;
        while (i < position) {
            tempNode = tempNode.next;
            i++;
        }
        return tempNode.next;
    }

    /**
     * 4、獲取連結串列的長度：時間複雜度為O(n)
     * @return
     */
    public int size() {
        tempNode = headNode.next;
        int size = 0;
        while (tempNode.next != null) {
            size = size + 1;
            tempNode = tempNode.next;
        }
        size = size + 1; // tempNode的地址域為null時，size記得加上最後一個節點
        return size;
    }

    // 遍歷連結串列，列印出所有節點的方法
    public void showAll() {
        tempNode = headNode.next;
        while (tempNode.next != null) {
            System.out.println(tempNode.element);
            tempNode = tempNode.next;
        }
        System.out.println(tempNode.element);
    }
```

```java
import org.junit.Test;

public class QuestionFourDemo {

    /**
     * 合併兩個有序連結串列，使合併之後的連結串列依然有序
     *
     * @param head1 有序連結串列1的第一個有效節點(注意與頭節點的區分！)
     * @param head2 有序連結串列2的第一個有效節點(注意與頭節點的區分！)
     * @return 返回合併好的有序連結串列的第一個有效節點(注意與頭節點的區分！)
     * @throws Exception
     */
    public NumberNode mergeLinkedList(NumberNode head1, NumberNode head2) throws Exception {
        // 如果兩個連結串列都為空
        if (head1 == null && head2 == null) {
            throw new Exception("兩個連結串列都為空");
        }
        if (head1 == null) {
            return head2;
        }
        if (head2 == null) {
            return head1;
        }

        // 新連結串列的第一個有效節點(注意與頭節點的區分！)
        NumberNode newHead;
        // temp指標會在兩個連結串列中來回選出較小的節點
        NumberNode temp;

        // 一開始，讓temp指標指向head1和head2中較小的資料，得到head節點
        if (head1.getElement() < head2.getElement()) {
            newHead = head1;
            temp = head1;
            head1 = head1.getNext();
        } else {
            newHead = head2;
            temp = head2;
            head2 = head2.getNext();
        }

        while (head1 != null && head2 != null) {
            if (head1.getElement() < head2.getElement()) {
                // temp指標的下一個節點對應較小的那個資料
                temp.setNext(head1);
                // temp指標往後移
                temp = temp.getNext();
                // head1也要往後移
                head1 = head1.getNext();
            } else {
                temp.setNext(head2);
                temp = temp.getNext();
                head2 = head2.getNext();
            }
        }

        // 合併剩下的節點，剩下的節點一定是都在同一個連結串列中
        // 如果head1不為空，說明連結串列1裡面還有節點，連結串列2已經被遍歷完了
        if (head1 != null) {
            temp.setNext(head1);
        }
        if (head2 != null) {
            temp.setNext(head2);
        }
        // 返回新連結串列的第一個有效節點(注意與頭節點的區分！)
        return newHead;
    }

    /**
     * 用來測試的方法
     */
    @Test
    public void test() throws Exception {
        MyNumberLinkedList list1 = new MyNumberLinkedList();
        MyNumberLinkedList list2 = new MyNumberLinkedList();

        // 向連結串列1中新增資料
        NumberNode node1_1 = new NumberNode(1, null);
        NumberNode node1_2 = new NumberNode(2, null);
        NumberNode node1_3 = new NumberNode(3, null);
        NumberNode node1_4 = new NumberNode(4, null);
        list1.insert(node1_1, 1);
        list1.insert(node1_2, 2);
        list1.insert(node1_3, 3);
        list1.insert(node1_4, 4);

        // 向連結串列2中新增資料
        NumberNode node2_2 = new NumberNode(2, null);
        NumberNode node2_3 = new NumberNode(3, null);
        NumberNode node2_4 = new NumberNode(4, null);
        NumberNode node2_5 = new NumberNode(5, null);
        list2.insert(node2_2, 1);
        list2.insert(node2_3, 2);
        list2.insert(node2_4, 3);
        list2.insert(node2_5, 4);

        // 分別輸出連結串列1和連結串列2
        System.out.println("連結串列1：");
        list1.showAll();
        System.out.println("");
        System.out.println("連結串列2：");
        list2.showAll();
        System.out.println("");

        // 合併之後輸出
        System.out.println("合併之後的連結串列：");
        NumberNode newNode = mergeLinkedList(list1.getHeadNode().getNext(), list2.getHeadNode().getNext());
        NumberNode newHeadNode = new NumberNode(null, newNode);
        MyNumberLinkedList newList = new MyNumberLinkedList();
        newList.setHeadNode(newHeadNode);
        newList.showAll();
    }
}
```

## 5、找出兩個單向連結串列相交的第一個公共節點

### 5-1：示意圖

**兩個節點相交的第一個公共節點如下圖：**

![][16]

### 5-2：思路

**方法一：**

面試時，很多人碰到這道題的第一反應是：在第一個連結串列上順序遍歷每個節點，每遍歷到一個節點的時候，在第二個連結串列上順序遍歷每個節點。如果在第二個連結串列上有一個節點和第一個連結串列上的節點一樣，說明兩個連結串列在這個節點上重合。所以這種方法的時間複雜度為 O(len1 * len2)  

**方法二：（用棧）**

參考上面的示意圖，如果兩個連結串列有公共節點，那麼最後一個節點（節點7）一定是一樣的，而且是從中間的某一個節點（節點6）開始，後面的節點都是一樣的。

現在的問題是，在單連結串列中，我們只能從頭節點開始順序遍歷，最後才能到達尾節點。最後到達的尾節點卻要先被比較，這聽起來是不是像“先進後出”？於是我們就能想到利用**棧**來解決這個問題：**分別把兩個連結串列的節點放入兩個棧中，這樣兩個連結串列的尾節點就位於兩個棧的棧頂，接下來從兩個棧的棧頂開始比較，直到找到最後一個相同的節點，就是他們的公共點。**

這種方法中，我們需要**利用兩個輔助棧**，空間複雜度是 O(len1+len2)，時間複雜度是 O(len1+len2)。跟方法一相比，時間效率得到了提高，相當於**利用空間來換取時間**。

那麼，有沒有更好的方法呢？接下來要講。

**方法三：（**快慢指標）****

其實我們還有一個更簡單的方法：**首先遍歷兩個連結串列得到它們的長度。在第二次遍歷的時候，在較長的連結串列上走 |len1-len2| 步，然後再同時遍歷這兩個連結串列，找到的第一個相同的節點就是它們的第一個公共點**。 

這種方法的時間複雜度也是 O(len1+len2)，但是我們不再需要用兩個棧，因此提高了空間效率。當面試官肯定了我們的最後一種思路的時候，就可以動手寫程式碼了。 

### 5-3：程式碼

```java
import org.junit.Test;

public class QuestionFiveDemo {

    /**
     * 方法：求兩個連結串列相交的第一個公共節點
     *
     * @param head1 連結串列1頭節點後面的第一個節點
     * @param head2 連結串列2頭節點後面的第一個節點
     * @return 返回兩個連結串列的第一個公共節點
     */
    public NumberNode getFirstCommonNode(NumberNode head1, NumberNode head2) {
        if (head1 == null || head2 == null) {
            return null;
        }

        int size1 = getSize(head1);
        int size2 = getSize(head2);
        // 兩個連結串列長度的差值
        int diffSize = 0;

        NumberNode longHead;
        NumberNode shortHead;

        // 找出較長的那個連結串列
        if (size1 > size2) {
            longHead = head1;
            shortHead = head2;
            diffSize = size1 - size2;
        } else {
            longHead = head2;
            shortHead = head1;
            diffSize = size2 - size1;
        }

        // 把較長的那個連結串列的指標向後移動diffSize個位置
        for (int i = 0; i < diffSize; i++) {
            longHead = longHead.getNext();
        }

        // 兩個連結串列的指標同時向後移動
        while (longHead != null && shortHead != null) {
            // 第一個相同的節點就是它們的公共節點
            if (longHead.getElement() == shortHead.getElement()) {
                return longHead;
            }
            longHead = longHead.getNext();
            shortHead = shortHead.getNext();
        }
        return null;
    }

    /**
     * 方法：獲取連結串列的長度
     *
     * @param head 指頭節點後面的第一個節點
     * @return 返回連結串列的長度
     */
    public int getSize(NumberNode head) {
        if (head == null) {
            return 0;
        }

        int size = 0;
        NumberNode temp = head;
        while (temp != null) {
            size++;
            temp = temp.getNext();
        }
        return size;
    }

    /**
     * 用來測試的方法
     */
    @Test
    public void test() throws Exception {
        MyNumberLinkedList list1 = new MyNumberLinkedList();
        MyNumberLinkedList list2 = new MyNumberLinkedList();

        // 向連結串列1中新增資料
        NumberNode node1_1 = new NumberNode(1, null);
        NumberNode node1_2 = new NumberNode(2, null);
        NumberNode node1_3 = new NumberNode(3, null);
        NumberNode node1_4 = new NumberNode(6, null);
        NumberNode node1_5 = new NumberNode(7, null);
        list1.insert(node1_1, 1);
        list1.insert(node1_2, 2);
        list1.insert(node1_3, 3);
        list1.insert(node1_4, 4);
        list1.insert(node1_5, 5);

        // 向連結串列2中新增資料
        NumberNode node2_4 = new NumberNode(4, null);
        NumberNode node2_5 = new NumberNode(5, null);
        NumberNode node2_6 = new NumberNode(6, null);
        NumberNode node2_7 = new NumberNode(7, null);
        list2.insert(node2_4, 1);
        list2.insert(node2_5, 2);
        list2.insert(node2_6, 3);
        list2.insert(node2_7, 4);

        // 分別輸出連結串列1和連結串列2
        System.out.println("連結串列1：");
        list1.showAll();
        System.out.println("");
        System.out.println("連結串列2：");
        list2.showAll();
        System.out.println("");

        // 輸出第一個公共節點
        NumberNode commonNode = getFirstCommonNode(node1_1, node2_4);
        System.out.println("第一個公共節點是：" + commonNode.getElement());
    }
}
```
