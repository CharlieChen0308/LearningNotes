# 04 樹與圖

> **版本**：Java 17+ — 所有程式碼基於 Java 17 編寫

樹與圖是電腦科學中最重要的非線性資料結構。樹用於表達層級關係（檔案系統、DOM、資料庫索引），圖則用於描述任意節點之間的連接（社交網路、地圖導航、網路拓撲）。本文涵蓋二叉樹遍歷、二叉搜尋樹、哈夫曼樹與圖的搜尋演算法，並提供完整可編譯的 Java 實作。

---

## 1、二叉樹（Binary Tree）

二叉樹是每個節點最多有兩個子節點（左子節點、右子節點）的樹狀結構。

### 節點定義

```java
public class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    public TreeNode(int val) {
        this.val = val;
    }
}
```

### 遍歷方式

二叉樹有四種基本遍歷方式，以下同時提供遞迴與迭代版本。

#### 前序遍歷（Pre-order）：根 → 左 → 右

```java
// 遞迴版
public void preorderRecursive(TreeNode root, List<Integer> result) {
    if (root == null) return;
    result.add(root.val);
    preorderRecursive(root.left, result);
    preorderRecursive(root.right, result);
}

// 迭代版（使用 Stack）
public List<Integer> preorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    if (root == null) return result;
    Deque<TreeNode> stack = new ArrayDeque<>();
    stack.push(root);
    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        result.add(node.val);
        // 先推右再推左，確保左子樹先被處理
        if (node.right != null) stack.push(node.right);
        if (node.left != null) stack.push(node.left);
    }
    return result;
}
```

#### 中序遍歷（In-order）：左 → 根 → 右

```java
// 遞迴版
public void inorderRecursive(TreeNode root, List<Integer> result) {
    if (root == null) return;
    inorderRecursive(root.left, result);
    result.add(root.val);
    inorderRecursive(root.right, result);
}

// 迭代版（使用 Stack）
public List<Integer> inorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode current = root;
    while (current != null || !stack.isEmpty()) {
        // 一路向左走到底
        while (current != null) {
            stack.push(current);
            current = current.left;
        }
        current = stack.pop();
        result.add(current.val);
        current = current.right;
    }
    return result;
}
```

#### 後序遍歷（Post-order）：左 → 右 → 根

```java
// 遞迴版
public void postorderRecursive(TreeNode root, List<Integer> result) {
    if (root == null) return;
    postorderRecursive(root.left, result);
    postorderRecursive(root.right, result);
    result.add(root.val);
}

// 迭代版（雙 Stack 法）
public List<Integer> postorderIterative(TreeNode root) {
    LinkedList<Integer> result = new LinkedList<>();
    if (root == null) return result;
    Deque<TreeNode> stack = new ArrayDeque<>();
    stack.push(root);
    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        // 將結果插入到最前面，形成 左→右→根 的順序
        result.addFirst(node.val);
        if (node.left != null) stack.push(node.left);
        if (node.right != null) stack.push(node.right);
    }
    return result;
}
```

#### 層序遍歷（Level-order / BFS）

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    while (!queue.isEmpty()) {
        int size = queue.size();
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
```

### 遍歷方式比較

| 遍歷方式 | 順序 | 常見用途 |
|---------|------|---------|
| 前序 | 根→左→右 | 複製樹、序列化 |
| 中序 | 左→根→右 | BST 取得排序結果 |
| 後序 | 左→右→根 | 刪除樹、計算目錄大小 |
| 層序 | 逐層由左至右 | 最短路徑、按層處理 |

## 2、二叉搜尋樹（BST）

二叉搜尋樹（Binary Search Tree）滿足以下性質：對於任意節點，**左子樹所有節點的值 < 該節點的值 < 右子樹所有節點的值**。對 BST 做中序遍歷可以得到有序序列。

### 搜尋

```java
public TreeNode search(TreeNode root, int target) {
    if (root == null || root.val == target) {
        return root;
    }
    if (target < root.val) {
        return search(root.left, target);
    } else {
        return search(root.right, target);
    }
}
```

### 插入

```java
public TreeNode insert(TreeNode root, int val) {
    if (root == null) {
        return new TreeNode(val);
    }
    if (val < root.val) {
        root.left = insert(root.left, val);
    } else if (val > root.val) {
        root.right = insert(root.right, val);
    }
    // val == root.val 時不重複插入
    return root;
}
```

### 刪除

刪除是 BST 中最複雜的操作，需要處理三種情況：

1. **葉節點**：直接刪除
2. **只有一個子節點**：用子節點替代
3. **有兩個子節點**：用右子樹的最小值（中序後繼）替代，再刪除該後繼節點

```java
public TreeNode delete(TreeNode root, int val) {
    if (root == null) return null;

    if (val < root.val) {
        root.left = delete(root.left, val);
    } else if (val > root.val) {
        root.right = delete(root.right, val);
    } else {
        // 找到目標節點
        if (root.left == null) return root.right;
        if (root.right == null) return root.left;
        // 有兩個子節點：找右子樹最小值
        TreeNode minNode = findMin(root.right);
        root.val = minNode.val;
        root.right = delete(root.right, minNode.val);
    }
    return root;
}

private TreeNode findMin(TreeNode node) {
    while (node.left != null) {
        node = node.left;
    }
    return node;
}
```

### 時間複雜度

| 操作 | 平均 | 最差（退化成鏈表） |
|------|------|------------------|
| 搜尋 | O(log n) | O(n) |
| 插入 | O(log n) | O(n) |
| 刪除 | O(log n) | O(n) |

為了避免最差情況，實務上會使用**自平衡 BST**：

- **AVL 樹**：嚴格平衡，任意節點左右子樹高度差不超過 1，適合讀多寫少的場景
- **紅黑樹**：近似平衡，插入和刪除效率更佳，Java 的 `TreeMap`、`TreeSet` 底層即採用紅黑樹

## 3、哈夫曼樹（Huffman Tree）

哈夫曼樹是一種**帶權路徑長度（WPL）最小**的二叉樹，廣泛應用於資料壓縮。其核心思想是：**出現頻率越高的字元，編碼越短**，且所有編碼互不為前綴（前綴碼），保證解碼的唯一性。

### 建構演算法

1. 將每個字元及其權重建立為葉節點，放入最小優先佇列
2. 每次取出權重最小的兩個節點，合併為新節點（權重為兩者之和）
3. 將新節點放回佇列
4. 重複直到佇列只剩一個節點，即為根節點

### Java 實作

```java
import java.util.*;

public class HuffmanTree {

    static class HuffmanNode implements Comparable<HuffmanNode> {
        int weight;
        char ch;       // 僅葉節點有意義
        HuffmanNode left;
        HuffmanNode right;

        public HuffmanNode(int weight, char ch) {
            this.weight = weight;
            this.ch = ch;
        }

        @Override
        public int compareTo(HuffmanNode other) {
            return Integer.compare(this.weight, other.weight);
        }
    }

    /** 建構哈夫曼樹 */
    public static HuffmanNode buildTree(Map<Character, Integer> freqMap) {
        PriorityQueue<HuffmanNode> pq = new PriorityQueue<>();
        for (var entry : freqMap.entrySet()) {
            pq.offer(new HuffmanNode(entry.getValue(), entry.getKey()));
        }
        while (pq.size() > 1) {
            HuffmanNode left = pq.poll();
            HuffmanNode right = pq.poll();
            HuffmanNode parent = new HuffmanNode(left.weight + right.weight, '\0');
            parent.left = left;
            parent.right = right;
            pq.offer(parent);
        }
        return pq.poll();
    }

    /** 產生哈夫曼編碼表 */
    public static Map<Character, String> generateCodes(HuffmanNode root) {
        Map<Character, String> codes = new HashMap<>();
        buildCodes(root, "", codes);
        return codes;
    }

    private static void buildCodes(HuffmanNode node, String prefix,
                                   Map<Character, String> codes) {
        if (node == null) return;
        // 葉節點：記錄編碼
        if (node.left == null && node.right == null) {
            codes.put(node.ch, prefix.isEmpty() ? "0" : prefix);
            return;
        }
        buildCodes(node.left, prefix + "0", codes);
        buildCodes(node.right, prefix + "1", codes);
    }

    public static void main(String[] args) {
        Map<Character, Integer> freq = Map.of(
            'A', 5, 'B', 9, 'C', 12, 'D', 13, 'E', 16, 'F', 45
        );

        HuffmanNode root = buildTree(freq);
        Map<Character, String> codes = generateCodes(root);

        System.out.println("字元\t頻率\t編碼");
        freq.entrySet().stream()
            .sorted(Map.Entry.comparingByValue())
            .forEach(e -> System.out.printf("%c\t%d\t%s%n",
                e.getKey(), e.getValue(), codes.get(e.getKey())));
    }
}
```

> **注意**：`compareTo()` 必須比較 `this.weight` 與 `other.weight`。若誤寫成 `that.value - that.value`（與自身比較），結果恆為 0，優先佇列無法正確排序，建構出的樹將不是最優解。使用 `Integer.compare()` 是最安全的寫法，可避免整數溢位問題。

## 4、圖（Graph）

圖由**頂點（Vertex）**和**邊（Edge）**組成，可以是有向或無向、有權或無權。

### 表示方式

| 方式 | 空間複雜度 | 查詢邊 | 適用場景 |
|------|----------|--------|---------|
| 鄰接矩陣 | O(V^2) | O(1) | 稠密圖、需快速判斷兩點是否相連 |
| 鄰接表 | O(V + E) | O(degree) | 稀疏圖、大多數實際應用 |

### 鄰接表實作

```java
import java.util.*;

public class Graph {
    private final int vertices;
    private final List<List<Integer>> adjList;

    public Graph(int vertices) {
        this.vertices = vertices;
        this.adjList = new ArrayList<>();
        for (int i = 0; i < vertices; i++) {
            adjList.add(new ArrayList<>());
        }
    }

    /** 新增無向邊 */
    public void addEdge(int u, int v) {
        adjList.get(u).add(v);
        adjList.get(v).add(u);
    }
```

### DFS（深度優先搜尋）

DFS 沿著一條路徑走到底，再回溯探索其他分支。

```java
    /** DFS — 遞迴版 */
    public List<Integer> dfsRecursive(int start) {
        List<Integer> result = new ArrayList<>();
        boolean[] visited = new boolean[vertices];
        dfsHelper(start, visited, result);
        return result;
    }

    private void dfsHelper(int node, boolean[] visited, List<Integer> result) {
        visited[node] = true;
        result.add(node);
        for (int neighbor : adjList.get(node)) {
            if (!visited[neighbor]) {
                dfsHelper(neighbor, visited, result);
            }
        }
    }

    /** DFS — 迭代版（使用 Stack） */
    public List<Integer> dfsIterative(int start) {
        List<Integer> result = new ArrayList<>();
        boolean[] visited = new boolean[vertices];
        Deque<Integer> stack = new ArrayDeque<>();
        stack.push(start);
        while (!stack.isEmpty()) {
            int node = stack.pop();
            if (visited[node]) continue;
            visited[node] = true;
            result.add(node);
            // 反向加入鄰居，使順序與遞迴版一致
            List<Integer> neighbors = adjList.get(node);
            for (int i = neighbors.size() - 1; i >= 0; i--) {
                if (!visited[neighbors.get(i)]) {
                    stack.push(neighbors.get(i));
                }
            }
        }
        return result;
    }
```

### BFS（廣度優先搜尋）

BFS 逐層向外擴展，適合尋找最短路徑（無權圖）。

```java
    /** BFS — 使用 Queue */
    public List<Integer> bfs(int start) {
        List<Integer> result = new ArrayList<>();
        boolean[] visited = new boolean[vertices];
        Queue<Integer> queue = new LinkedList<>();
        visited[start] = true;
        queue.offer(start);
        while (!queue.isEmpty()) {
            int node = queue.poll();
            result.add(node);
            for (int neighbor : adjList.get(node)) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    queue.offer(neighbor);
                }
            }
        }
        return result;
    }
```

### 使用範例

```java
    public static void main(String[] args) {
        Graph g = new Graph(6);
        g.addEdge(0, 1);
        g.addEdge(0, 2);
        g.addEdge(1, 3);
        g.addEdge(2, 4);
        g.addEdge(3, 5);
        g.addEdge(4, 5);

        System.out.println("DFS（遞迴）: " + g.dfsRecursive(0));
        System.out.println("DFS（迭代）: " + g.dfsIterative(0));
        System.out.println("BFS:         " + g.bfs(0));
    }
}
```

### DFS vs BFS 比較

| 特性 | DFS | BFS |
|------|-----|-----|
| 資料結構 | Stack（或遞迴） | Queue |
| 空間複雜度 | O(h)，h 為深度 | O(w)，w 為最大寬度 |
| 最短路徑 | 不保證 | 保證（無權圖） |
| 適用場景 | 路徑搜尋、拓撲排序、連通分量 | 最短路徑、層級遍歷 |

## 5、常見應用場景

| 資料結構 | 典型應用 |
|---------|---------|
| 二叉搜尋樹 | 有序集合（`TreeSet`）、字典（`TreeMap`）、資料庫索引 |
| 哈夫曼樹 | 資料壓縮（ZIP、GZIP、JPEG 中的熵編碼） |
| 圖 DFS | 拓撲排序、迷宮求解、連通分量偵測、迴圈偵測 |
| 圖 BFS | 最短路徑（無權圖）、社交網路好友推薦、網頁爬蟲 |

## 6、小結

本文介紹了樹與圖兩大非線性資料結構的核心知識：

- **二叉樹遍歷**是所有樹狀結構操作的基礎，遞迴寫法直觀，迭代寫法在面試中更受青睞
- **二叉搜尋樹**提供 O(log n) 的查找效率，但需注意退化問題；實務上使用自平衡變體（Java `TreeMap` 底層為紅黑樹）
- **哈夫曼樹**展示了貪心演算法的經典應用，實作時務必確認 `compareTo()` 的正確性
- **圖的 DFS 與 BFS** 是幾乎所有圖演算法的基石，掌握後可進一步學習 Dijkstra、Bellman-Ford、Kruskal 等進階演算法
