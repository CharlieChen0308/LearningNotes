# 04 Vue 3 元件開發實戰

> **版本**：Vue 3.4+ / Vue Router 4 / Element Plus / Axios

本篇聚焦實戰：如何用 Vue 3 + TypeScript 建構一個完整的管理系統前端，涵蓋 SFC 結構、UI 框架整合、路由、API 封裝、表單驗證與表格分頁。

---

## 1、SFC 結構

Vue 的單檔元件（Single File Component, SFC）將模板、邏輯、樣式集中在一個 `.vue` 檔案中。搭配 `<script setup>` 語法糖，寫法最為簡潔：

```vue
<script setup lang="ts">
import { ref, computed } from "vue"

interface Tire {
  id: number
  brand: string
  model: string
  price: number
}

const tire = ref<Tire>({
  id: 1,
  brand: "Michelin",
  model: "Pilot Sport 5",
  price: 4200
})

const displayPrice = computed(() => `NT$ ${tire.value.price.toLocaleString()}`)
</script>

<template>
  <div class="tire-card">
    <h3>{{ tire.brand }} {{ tire.model }}</h3>
    <span class="price">{{ displayPrice }}</span>
  </div>
</template>

<style scoped>
.tire-card {
  padding: 16px;
  border: 1px solid #e4e7ed;
  border-radius: 8px;
}

.price {
  color: #e6a23c;
  font-weight: bold;
}
</style>
```

**三個區塊的職責**：

- `<script setup lang="ts">`：元件邏輯，所有頂層變數和函式自動暴露給模板
- `<template>`：HTML 模板，使用 Vue 的模板語法
- `<style scoped>`：`scoped` 確保樣式只作用於當前元件，不會污染其他元件

## 2、Element Plus 整合

Element Plus 是 Vue 3 最受歡迎的 UI 元件庫，提供豐富的表單、表格、對話框等元件。

### 安裝

```bash
npm install element-plus
npm install -D unplugin-vue-components unplugin-auto-import
```

### 按需引入（推薦）

在 `vite.config.ts` 中設定自動按需引入，避免打包整個元件庫：

```typescript
// vite.config.ts
import { defineConfig } from "vite"
import vue from "@vitejs/plugin-vue"
import AutoImport from "unplugin-auto-import/vite"
import Components from "unplugin-vue-components/vite"
import { ElementPlusResolver } from "unplugin-vue-components/resolvers"

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()]
    }),
    Components({
      resolvers: [ElementPlusResolver()]
    })
  ]
})
```

設定完成後，直接在模板中使用 Element Plus 元件，無需手動 import：

```vue
<template>
  <el-button type="primary">主要按鈕</el-button>
  <el-input v-model="keyword" placeholder="搜尋..." />
</template>
```

### 常用元件速查

| 元件 | 用途 | 關鍵屬性 |
|------|------|---------|
| `ElTable` + `ElTableColumn` | 資料表格 | `:data`, `prop`, `label` |
| `ElForm` + `ElFormItem` | 表單容器 | `:model`, `:rules`, `prop` |
| `ElDialog` | 對話框 | `v-model`, `title`, `width` |
| `ElMessage` | 訊息提示 | `ElMessage.success()` / `.error()` |
| `ElPagination` | 分頁 | `total`, `page-size`, `current-page` |
| `ElSelect` + `ElOption` | 下拉選單 | `v-model`, `label`, `value` |

## 3、路由（Vue Router 4）

### 安裝與基本設定

```bash
npm install vue-router@4
```

```typescript
// router/index.ts
import { createRouter, createWebHistory, type RouteRecordRaw } from "vue-router"

const routes: RouteRecordRaw[] = [
  {
    path: "/",
    redirect: "/dashboard"
  },
  {
    path: "/login",
    name: "Login",
    component: () => import("@/views/LoginView.vue")
  },
  {
    path: "/",
    component: () => import("@/layouts/MainLayout.vue"),
    children: [
      {
        path: "dashboard",
        name: "Dashboard",
        component: () => import("@/views/DashboardView.vue")
      },
      {
        path: "tires",
        name: "TireList",
        component: () => import("@/views/tire/TireListView.vue")
      },
      {
        path: "tires/:id",
        name: "TireDetail",
        component: () => import("@/views/tire/TireDetailView.vue"),
        props: true  // 將 :id 作為 prop 傳入元件
      }
    ]
  }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

### 動態路由

```vue
<!-- TireDetailView.vue -->
<script setup lang="ts">
import { useRoute, useRouter } from "vue-router"
import { onMounted, ref } from "vue"

// 方式一：透過 props（需在路由定義 props: true）
const props = defineProps<{ id: string }>()

// 方式二：透過 useRoute()
const route = useRoute()
const tireId = route.params.id

// 程式化導航
const router = useRouter()
function goBack() {
  router.push({ name: "TireList" })
}

function goToTire(id: number) {
  router.push({ name: "TireDetail", params: { id } })
}
</script>
```

### 導航守衛（Auth Check）

```typescript
// router/index.ts
import { useAuthStore } from "@/stores/authStore"

router.beforeEach((to, from, next) => {
  const authStore = useAuthStore()

  // 不需要認證的頁面
  if (to.name === "Login") {
    next()
    return
  }

  // 檢查是否已登入
  if (!authStore.isAuthenticated) {
    next({ name: "Login", query: { redirect: to.fullPath } })
    return
  }

  next()
})
```

### 巢狀路由

巢狀路由透過 `children` 陣列定義，父元件需要包含 `<router-view>` 來顯示子路由：

```vue
<!-- layouts/MainLayout.vue -->
<script setup lang="ts">
// 主布局：側邊欄 + 頂部導航 + 內容區
</script>

<template>
  <div class="layout">
    <aside class="sidebar">
      <router-link to="/dashboard">儀表板</router-link>
      <router-link to="/tires">輪胎管理</router-link>
    </aside>
    <main class="content">
      <router-view />  <!-- 子路由在這裡渲染 -->
    </main>
  </div>
</template>
```

## 4、Axios 封裝

### 建立 Axios Instance

```typescript
// utils/request.ts
import axios, { type AxiosInstance, type AxiosResponse, type InternalAxiosRequestConfig } from "axios"
import { ElMessage } from "element-plus"
import router from "@/router"

const service: AxiosInstance = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL || "/api",
  timeout: 15000
})
```

### 請求攔截器（加 JWT Token）

```typescript
// utils/request.ts（接續上方）
service.interceptors.request.use(
  (config: InternalAxiosRequestConfig) => {
    const token = localStorage.getItem("token")
    if (token) {
      config.headers.Authorization = `Bearer ${token}`
    }
    return config
  },
  (error) => {
    return Promise.reject(error)
  }
)
```

### 回應攔截器（統一錯誤處理）

```typescript
// utils/request.ts（接續上方）
service.interceptors.response.use(
  (response: AxiosResponse) => {
    return response.data
  },
  (error) => {
    const status = error.response?.status
    const message = error.response?.data?.message || "請求失敗"

    switch (status) {
      case 401:
        ElMessage.error("登入已過期，請重新登入")
        localStorage.removeItem("token")
        router.push({ name: "Login" })
        break
      case 403:
        ElMessage.error("權限不足")
        break
      case 404:
        ElMessage.error("資源不存在")
        break
      case 500:
        ElMessage.error("伺服器錯誤")
        break
      default:
        ElMessage.error(message)
    }

    return Promise.reject(error)
  }
)

export default service
```

### API 模組化

```typescript
// api/tire.ts
import request from "@/utils/request"

export interface Tire {
  id: number
  brand: string
  model: string
  size: number
  price: number
}

export interface TireQuery {
  keyword?: string
  category?: string
  page: number
  pageSize: number
}

export interface PageResult<T> {
  items: T[]
  total: number
}

// 查詢輪胎列表（分頁）
export function getTires(params: TireQuery): Promise<PageResult<Tire>> {
  return request.get("/tires", { params })
}

// 取得單一輪胎
export function getTireById(id: number): Promise<Tire> {
  return request.get(`/tires/${id}`)
}

// 新增輪胎
export function createTire(data: Omit<Tire, "id">): Promise<Tire> {
  return request.post("/tires", data)
}

// 更新輪胎
export function updateTire(id: number, data: Partial<Tire>): Promise<Tire> {
  return request.put(`/tires/${id}`, data)
}

// 刪除輪胎
export function deleteTire(id: number): Promise<void> {
  return request.delete(`/tires/${id}`)
}
```

## 5、表單與驗證

### Element Plus ElForm + Rules

```vue
<script setup lang="ts">
import { ref, reactive } from "vue"
import type { FormInstance, FormRules } from "element-plus"
import { ElMessage } from "element-plus"
import { createTire } from "@/api/tire"

// 表單 ref（用於呼叫 validate）
const formRef = ref<FormInstance>()

// 表單資料
const form = reactive({
  brand: "",
  model: "",
  size: undefined as number | undefined,
  price: undefined as number | undefined
})

// 驗證規則
const rules: FormRules = {
  brand: [
    { required: true, message: "請輸入品牌", trigger: "blur" }
  ],
  model: [
    { required: true, message: "請輸入型號", trigger: "blur" },
    { min: 2, max: 50, message: "長度 2 ~ 50 字", trigger: "blur" }
  ],
  size: [
    { required: true, message: "請輸入尺寸", trigger: "blur" }
  ],
  price: [
    { required: true, message: "請輸入價格", trigger: "blur" },
    { validator: validatePrice, trigger: "blur" }
  ]
}

// 自定義驗證器
function validatePrice(rule: any, value: number, callback: (error?: Error) => void) {
  if (value <= 0) {
    callback(new Error("價格必須大於 0"))
  } else if (value > 100000) {
    callback(new Error("價格不可超過 100,000"))
  } else {
    callback()
  }
}

// 提交處理
async function handleSubmit() {
  // 先驗證表單
  const valid = await formRef.value?.validate().catch(() => false)
  if (!valid) return

  try {
    await createTire({
      brand: form.brand,
      model: form.model,
      size: form.size!,
      price: form.price!
    })
    ElMessage.success("新增成功")
    resetForm()
  } catch (error) {
    // 錯誤已由攔截器統一處理
  }
}

// 重置表單
function resetForm() {
  formRef.value?.resetFields()
}
</script>

<template>
  <el-form ref="formRef" :model="form" :rules="rules" label-width="80px">
    <el-form-item label="品牌" prop="brand">
      <el-input v-model="form.brand" placeholder="請輸入品牌" />
    </el-form-item>

    <el-form-item label="型號" prop="model">
      <el-input v-model="form.model" placeholder="請輸入型號" />
    </el-form-item>

    <el-form-item label="尺寸" prop="size">
      <el-input-number v-model="form.size" :min="12" :max="24" />
    </el-form-item>

    <el-form-item label="價格" prop="price">
      <el-input-number v-model="form.price" :min="0" :step="100" />
    </el-form-item>

    <el-form-item>
      <el-button type="primary" @click="handleSubmit">新增</el-button>
      <el-button @click="resetForm">重置</el-button>
    </el-form-item>
  </el-form>
</template>
```

**關鍵要點**：

- `formRef` 透過 `ref<FormInstance>()` 取得表單實例
- `:model` 綁定表單資料物件
- `:rules` 綁定驗證規則，每個 `prop` 對應規則中的 key
- `validate()` 回傳 Promise，可搭配 `async/await` 使用
- 自定義驗證器透過 `validator` 屬性指定，回傳 `callback()` 表示通過，`callback(new Error())` 表示失敗

## 6、表格與分頁

### ElTable + ElPagination + 後端分頁

```vue
<script setup lang="ts">
import { ref, reactive, onMounted } from "vue"
import { ElMessage, ElMessageBox } from "element-plus"
import { getTires, deleteTire, type Tire, type TireQuery } from "@/api/tire"

// 表格資料
const tableData = ref<Tire[]>([])
const loading = ref(false)

// 分頁參數
const pagination = reactive({
  currentPage: 1,
  pageSize: 10,
  total: 0
})

// 搜尋參數
const searchForm = reactive({
  keyword: "",
  category: ""
})

// 載入資料
async function loadData() {
  loading.value = true
  try {
    const query: TireQuery = {
      keyword: searchForm.keyword || undefined,
      category: searchForm.category || undefined,
      page: pagination.currentPage,
      pageSize: pagination.pageSize
    }
    const result = await getTires(query)
    tableData.value = result.items
    pagination.total = result.total
  } catch (error) {
    // 錯誤已由攔截器統一處理
  } finally {
    loading.value = false
  }
}

// 換頁
function handlePageChange(page: number) {
  pagination.currentPage = page
  loadData()
}

// 變更每頁筆數
function handleSizeChange(size: number) {
  pagination.pageSize = size
  pagination.currentPage = 1  // 回到第一頁
  loadData()
}

// 搜尋
function handleSearch() {
  pagination.currentPage = 1
  loadData()
}

// 刪除（含確認對話框）
async function handleDelete(row: Tire) {
  try {
    await ElMessageBox.confirm(
      `確定要刪除「${row.brand} ${row.model}」嗎？`,
      "確認刪除",
      { type: "warning" }
    )
    await deleteTire(row.id)
    ElMessage.success("刪除成功")
    loadData()
  } catch {
    // 使用者取消，不做任何事
  }
}

onMounted(() => loadData())
</script>

<template>
  <div class="tire-list">
    <!-- 搜尋列 -->
    <div class="search-bar">
      <el-input
        v-model="searchForm.keyword"
        placeholder="搜尋品牌或型號"
        clearable
        style="width: 240px"
        @keyup.enter="handleSearch"
      />
      <el-select
        v-model="searchForm.category"
        placeholder="分類"
        clearable
        style="width: 160px; margin-left: 12px"
      >
        <el-option label="夏季胎" value="summer" />
        <el-option label="冬季胎" value="winter" />
        <el-option label="四季胎" value="all_season" />
      </el-select>
      <el-button type="primary" style="margin-left: 12px" @click="handleSearch">
        搜尋
      </el-button>
    </div>

    <!-- 資料表格 -->
    <el-table :data="tableData" v-loading="loading" stripe border style="width: 100%">
      <el-table-column prop="id" label="ID" width="80" />
      <el-table-column prop="brand" label="品牌" width="150" />
      <el-table-column prop="model" label="型號" min-width="200" />
      <el-table-column prop="size" label="尺寸" width="100" align="center">
        <template #default="{ row }">
          {{ row.size }} 吋
        </template>
      </el-table-column>
      <el-table-column prop="price" label="價格" width="140" align="right">
        <template #default="{ row }">
          NT$ {{ row.price.toLocaleString() }}
        </template>
      </el-table-column>
      <el-table-column label="操作" width="180" align="center" fixed="right">
        <template #default="{ row }">
          <el-button size="small" @click="$router.push(`/tires/${row.id}`)">
            檢視
          </el-button>
          <el-button size="small" type="danger" @click="handleDelete(row)">
            刪除
          </el-button>
        </template>
      </el-table-column>
    </el-table>

    <!-- 分頁 -->
    <div class="pagination-wrapper">
      <el-pagination
        v-model:current-page="pagination.currentPage"
        v-model:page-size="pagination.pageSize"
        :total="pagination.total"
        :page-sizes="[10, 20, 50]"
        layout="total, sizes, prev, pager, next, jumper"
        @current-change="handlePageChange"
        @size-change="handleSizeChange"
      />
    </div>
  </div>
</template>

<style scoped>
.search-bar {
  display: flex;
  align-items: center;
  margin-bottom: 16px;
}

.pagination-wrapper {
  display: flex;
  justify-content: flex-end;
  margin-top: 16px;
}
</style>
```

**關鍵要點**：

- `v-loading` 是 Element Plus 提供的載入指令，綁定 boolean 控制顯示
- `#default="{ row }"` 是 scoped slot 語法，用於自定義欄位內容
- 搜尋和換頁都重新呼叫 `loadData()`，搜尋時回到第一頁
- 刪除操作先用 `ElMessageBox.confirm` 確認，避免誤刪
- 分頁參數（`page`、`pageSize`）傳給後端，由後端負責分頁查詢

## 7、小結

本篇涵蓋了 Vue 3 前端開發的主要實戰技術：

- **SFC 結構**：`<script setup>` + `<template>` + `<style scoped>` 三段式組織
- **Element Plus**：搭配 `unplugin-vue-components` 按需引入，常用 ElTable、ElForm、ElDialog、ElMessage
- **Vue Router 4**：`createRouter` + `createWebHistory`，動態路由 `:id`，`beforeEach` 導航守衛，巢狀路由
- **Axios 封裝**：建立 instance，請求攔截器加 JWT，回應攔截器統一錯誤處理，API 模組化
- **表單驗證**：ElForm + rules + 自定義驗證器，`validate()` 搭配 async/await
- **表格分頁**：ElTable + ElPagination，後端分頁整合，搜尋重置頁碼
