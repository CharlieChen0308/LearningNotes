## 1、什麼是佇列 

（1）首先，佇列也是一種特殊的[線性表](http://www.cnblogs.com/nnngu/p/8247210.html)，它是一種操作受限的線性表。只允許在表的一端進行元素插入，而在另一端進行元素刪除。允許插入的一端稱為隊尾，允許刪除的一端稱為隊頭。

（2）佇列與現實生活中的排隊類似（如下圖），新加入的成員總是在隊尾，而排在佇列最前面的總是最先離開佇列，即先進先出 First In First Out (FIFO)，因此佇列就是先進先出線性表。

![][1]

 （3）線性表分為順序表和連結串列，所以佇列也分為**順序佇列**和**鏈式佇列，**為了方便演示，下文所使用的佇列都是**順序佇列**。

## 2、佇列的儲存結構

![][2]

用java語言自己封裝一個順序佇列：

SeqQueue.java

<pre>/**
 * 封裝一個順序佇列
 */
public class SeqQueue {
    // 儲存資料
    public Object[] data;

    // 頭指標
    public int head;

    // 尾指標
    public int rear;

    // 佇列的最大容量
    public int maxSize;

    public SeqQueue(int maxSize) {
        this.maxSize = maxSize;
        data = new Object[maxSize];
    }
}</pre>

## 3、佇列的常用操作及實現程式碼

### 3-1、初始化佇列

思路：構造一個空佇列，並將頭指標head和尾指標rear都設定為0。

程式碼：SeqQueueOperate.java

<pre>/**
 * 封裝佇列的常見操作
 */
public class SeqQueueOperate {
    /**
     * 初始化
     *
     * @param maxSize
     * @return
     */
    public SeqQueue init(int maxSize) {
        SeqQueue queue = new SeqQueue(maxSize);
        queue.head = 0;
        queue.rear = 0;
        return queue;
    }
}</pre>

### 3-2、入隊

思路：若佇列沒有滿，則將資料插入到尾指標rear指向的位置，然後再將rear加1。

程式碼：在 SeqQueueOperate.java 中新增方法

<pre>    /**
     * 入隊
     */
    public void enter(SeqQueue queue, Object obj) {
        // 判斷佇列是否已經滿了
        if (queue.rear >= queue.maxSize) {
            return;
        }
        queue.data[queue.rear] = obj;
        queue.rear++;
    }</pre>

### 3-3、出隊

思路：若佇列不為空，則將頭指標指向的元素刪除，然後將頭指標加1。

程式碼：在 SeqQueueOperate.java 中新增方法

<pre>    /**
     * 出隊
     */
    public Object dequeue(SeqQueue queue) {
        // 判斷佇列是否為空
        if (queue.head == queue.rear) {
            return null;
        }
        Object obj = queue.data[queue.head];
        queue.data[queue.head] = null;
        queue.head++;
        return obj;
    }</pre>

### 3-4、取隊頭

思路：若佇列不為空，則返回隊頭元素。

程式碼：在 SeqQueueOperate.java 中新增方法

<pre>    /**
     * 取隊頭
     */
    public Object getHead(SeqQueue queue) {
        // 判斷佇列是否為空
        if (queue.head == queue.rear) {
            return null;
        }
        Object obj = queue.data[queue.head];
        return obj;
    }</pre>

### 3-5、取隊長

思路：即尾指標 - 頭指標的值。

程式碼：在 SeqQueueOperate.java 中新增方法

<pre>    /**
     * 取隊長
     */
    public int getLength(SeqQueue queue) {
        return queue.rear - queue.head;
    }</pre>

### 3-6、判隊空

思路：只需要判斷頭指標和尾指標是否相等即可

程式碼：在 SeqQueueOperate.java 中新增方法

<pre>    /**
     * 判斷佇列是否為空
     */
    public boolean isEmpty(SeqQueue queue) {
        return queue.head == queue.rear;
    }</pre>

### 3-7、判隊滿

思路：只需判斷尾指標與maxSize是否相等即可

程式碼：在 SeqQueueOperate.java 中新增方法

<pre>    /**
     * 判斷佇列是否已經滿了
     */
    public boolean isFull(SeqQueue queue) {
        return queue.rear >= queue.maxSize;
    }</pre>

## 4、測試

新增一個用來測試的類

QueueTest.java

<pre>/**
 * 用來測試
 */
public class QueueTest {
    public static void main(String[] args) {
        SeqQueueOperate seqQueueOperate = new SeqQueueOperate();
        // 最大容量設定為5
        int maxSize = 5;
        SeqQueue queue = seqQueueOperate.init(maxSize);
        System.out.println("佇列的最大容量是：" + maxSize);

        // 當前佇列的長度
        System.out.println("當前佇列的長度是：" + seqQueueOperate.getLength(queue));
        System.out.println("");

        System.out.println("===========入隊start ===========");
        System.out.println("插入6個元素試試");
        seqQueueOperate.enter(queue, 1);
        seqQueueOperate.enter(queue, 2);
        seqQueueOperate.enter(queue, 3);
        seqQueueOperate.enter(queue, 4);
        seqQueueOperate.enter(queue, 5);
        seqQueueOperate.enter(queue, 6);
        System.out.println("===========入隊end =============");
        // 當前佇列的長度
        System.out.println("當前佇列的長度是：" + seqQueueOperate.getLength(queue));
        System.out.println("");

        // 出隊
        System.out.println("===========出隊start ===========");
        Object obj = seqQueueOperate.dequeue(queue);
        System.out.println("出隊的元素是：" + obj);
        System.out.println("===========出隊end =============");
        // 當前佇列的長度
        System.out.println("當前佇列的長度是：" + seqQueueOperate.getLength(queue));
        System.out.println("");

        System.out.println("------------------------------------");
        System.out.println("隊頭元素是：" + queue.data[queue.head]);
        System.out.println("隊尾元素是：" + queue.data[queue.rear-1]);
    }
}</pre>
