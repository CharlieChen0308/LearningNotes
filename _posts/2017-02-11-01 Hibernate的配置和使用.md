## Hibernate 介紹

Hibernate 是一個基於 jdbc 的開源的持久化框架，是一個優秀的 ORM 實現，它很大程度的簡化了 dao 層的編碼工作。

Hibernate 對 JDBC 訪問資料庫的程式碼做了封裝，大大簡化了資料訪問層繁瑣的重複性程式碼。 

Hibernate 在分層結構中處於持久化層，封裝對資料庫的訪問細節，使程式設計師更專注於實現業務邏輯。

## Hibernate 的開發步驟

1. 建立持久化類

3. 建立 Hibernate 配置檔案  （ hibernate.cfg.xml ）
4. 透過 Hibernate API 編寫訪問資料庫的程式碼

### 建立持久化類

在 MySQL 資料庫建立一個 t_user 表，建表語句如下：

```sql
create table t_user( id int primary key auto_increment, name varchar(20) );
```

建立一個持久化類 `User.java` ，程式碼如下：

```java
public class User {
    private int id;
    private String name;

    public int getId() {
        return id;
    }

    public void setId(int id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    @Override
    public String toString() {
        return "[User: id=" + id + ", name=" + name + "]";
    }
}

```

### 建立 物件-關係 對映檔案  （ \*.hbm.xml ）

建立對映檔案 `User.hbm.xml`

**注意：該對映檔案要與 User 類放在同一個包下。**

**如果使用 Maven 構建的專案，所有的配置檔案統一放在 resources 目錄下。因為 Maven 預設只會讀取 resources 目錄下的配置檔案。**

![][3]

 `User.hbm.xml`的程式碼如下：
 
 ```xml
 <?xml version="1.0"?>
<!DOCTYPE hibernate-mapping PUBLIC
        "-//Hibernate/Hibernate Mapping DTD 3.0//EN"
        "http://www.hibernate.org/dtd/hibernate-mapping-3.0.dtd">

<hibernate-mapping package="com.nnngu.entity">
    <class name="User" table="t_user">
        <id name="id" type="int" column="id">
            <generator class="native"/>
        </id>
        <property name="name" type="string" column="name"/>
    </class>
</hibernate-mapping>
 ```
 
### 建立Hibernate配置檔案（hibernate.cfg.xml）
 
 在 resources 目錄下建立 `hibernate.cfg.xml` 
 
 ![][4]
 
  `hibernate.cfg.xml` 的程式碼如下：
  
  ```xml
<!DOCTYPE hibernate-configuration PUBLIC
        "-//Hibernate/Hibernate Configuration DTD 3.0//EN"
        "http://www.hibernate.org/dtd/hibernate-configuration-3.0.dtd">

<hibernate-configuration>
    <session-factory>
        <!-- 1. 配置資料庫資訊 -->
        <!-- 方言(連線的資料庫型別) -->

        <property name="connection.username">root</property>
        <property name="connection.password">1</property>

        <!-- 2. 其他配置 -->
        <!-- 顯示生成的SQL語句 -->
        <property name="hibernate.show_sql">true</property>

        <!-- 3. 匯入對映檔案 -->
        <mapping resource="com.nnngu.entity/User.hbm.xml"/>
    </session-factory>
</hibernate-configuration>  
  ```

### 透過Hibernate API編寫訪問資料庫的程式碼
  
步驟：

1. 獲取 Configuration 物件
2. 獲取 SessionFactory 物件
3. 獲取 Session ，開啟事務
4. 用物件導向的方式運算元據庫
5. 提交事務，關閉 Session
  
首先建立一個工具類 `HibernateUtils.java`，該工具類的作用：

```
① 用來獲取 Configuration 物件
② 獲取全域性唯一的 SessionFactory 
③ 從全域性唯一的 SessionFactory 中開啟一個 Session 
```

`HibernateUtils.java`的程式碼如下：

```java
package com.nnngu.dao;

import org.hibernate.Session;
import org.hibernate.SessionFactory;
import org.hibernate.cfg.Configuration;

public class HibernateUtils {

    // SessionFactory全域性只需要有一個就可以了，因為它的建立和銷燬需要消耗大量的資源。而且它是執行緒安全的，可以在多執行緒的環境下使用它
    private static SessionFactory sessionFactory;

    static {
        // 初始化SessionFactory
        sessionFactory = new Configuration() // 代表配置檔案的一個物件
                .configure() // 讀取預設的配置檔案(hibernate.cfg.xml)
                .buildSessionFactory();
    }

    /**
     * 獲取全域性唯一的SessionFactory
     *
     * @return
     */
    public static SessionFactory getSessionFactory() {
        return sessionFactory;
    }

    public static Session openSession() {
        return sessionFactory.openSession();
    }
}

```

然後建立一個 UserDao 類，裡面編寫程式碼實現對資料庫的操作 (CRUD) 。

```java
package com.nnngu.dao;

import com.nnngu.entity.User;
import org.hibernate.Session;
import org.hibernate.Transaction;

import java.util.List;

public class UserDao {

    /**
     * 增
     *
     * @param user
     */
    public void save(User user) {
        Session session = HibernateUtils.openSession();
        Transaction tx = null;
        try {
            tx = session.beginTransaction(); // 開啟事務
            session.save(user); // 運算元據庫
            tx.commit(); // 提交事務
        } catch (RuntimeException e) {
            tx.rollback(); // 回滾事務
            throw e;
        } finally {
            session.close(); // 關閉session
        }
    }

    /**
     * 刪
     *
     * @param id
     */
    public void delete(int id) {
        Session session = HibernateUtils.openSession();
        Transaction tx = null;
        try {
            tx = session.beginTransaction();
            Object user = session.get(User.class, id); // 先獲取到這個物件
            session.delete(user); // 刪除
            tx.commit();
        } catch (RuntimeException e) {
            tx.rollback();
            throw e;
        } finally {
            session.close();
        }
    }

    /**
     * 改
     *
     * @param user
     */
    public void update(User user) {
        Session session = HibernateUtils.openSession();
        Transaction tx = null;
        try {
            tx = session.beginTransaction();
            session.update(user); // 運算元據庫
            tx.commit();
        } catch (RuntimeException e) {
            tx.rollback();
            throw e;
        } finally {
            session.close();
        }
    }

    /**
     * 查，根據id查詢
     *
     * @param id
     * @return
     */
    public User getById(int id) {
        Session session = HibernateUtils.openSession();
        Transaction tx = null;
        try {
            tx = session.beginTransaction();
            User user = (User) session.get(User.class, id); // 運算元據庫
            tx.commit();
            return user;
        } catch (RuntimeException e) {
            tx.rollback();
            throw e;
        } finally {
            session.close();
        }
    }

    /**
     * 查，查詢所有
     *
     * @return
     */
    public List<User> findAll() {
        Session session = HibernateUtils.openSession();
        Transaction tx = null;
        try {
            tx = session.beginTransaction();
            // 使用HQL語句
            List<User> list = session.createQuery("from User").list();
            tx.commit();
            return list;
        } catch (RuntimeException e) {
            tx.rollback();
            throw e;
        } finally {
            session.close();
        }
    }
}

```

## 測試

編寫測試類 `UserDaoTest.java`

程式碼如下：

```java
package com.nnngu.test;

import com.nnngu.dao.UserDao;
import com.nnngu.entity.User;
import org.junit.Test;

import java.util.List;

public class UserDaoTest {
    private UserDao userDao = new UserDao();

    @Test
    /**
     * 增
     */
    public void testSave() {
        User user = new User();
        user.setName("黃藥師");
        // 儲存
        userDao.save(user);
    }

    @Test
    /**
     * 刪
     */
    public void testDelete() {
        userDao.delete(1);
    }

    @Test
    /**
     * 改
     */
    public void testUpdate() {
        // 從資料庫中獲取一條存在的資料
        User user = userDao.getById(2);
        user.setName("東邪");
        // 更新
        userDao.update(user);
    }

    @Test
    /**
     * 查，根據id查詢
     */
    public void testGetById() {
        User user = userDao.getById(2);
        System.out.println(user);
    }

    @Test
    /**
     * 查，查詢所有
     */
    public void testFindAll() {
        List<User> list = userDao.findAll();
        for (User user : list) {
            System.out.println(user);
        }
    }
}
```
